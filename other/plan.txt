ЗАДАНИЕ
Организовать одностороннюю очередь. При запуске с помощью циклического сдвига вправо (fifo)
преобразовать очередь так, чтобы в нулевом индексе стоял наибольший элемент очереди.
Сохранить результат в файл

Формат данных: вещественные

РЕАЛИЗАЦИЯ
Класс OnewayQueue (односторонняя очередь)
Класс QueueIterator (для перебора очереди и циклического сдвига, реализует интерфейс Iterator)
Класс ResultSaver (для сохранения потока в файл)
Класс UI (взаимодействие с пользователем)
Класс Main (в котором происходит метод ui.run())

OnewayQueue
- приватное поле односторонней очереди

- вывод очереди
-

QueueIterator
- приватное поле итератора через метод .iterator()

- проверка на следующее значение

ResultSaver
- приватное поле потока

- чтение файла
- сохранение файла

UI
- метод run()

Main
- запуск UI
